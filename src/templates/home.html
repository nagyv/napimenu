About

Hi, this is a test app similar to TodoMVC. The main difference between the two is that we've tried to come up with a still simple, but already a bit more demanding requirements, like related models and several views on a page.

We think that the apps we have to build with these JS frameworks definitely need these requirements.

Where can I find the apps?

Our project is a work in progress. The current implementation are the following:

  • Non-js site
  • Pure jQuery
  • Angular JS
  • Backbone with Marionette
  • Knockout JS
Our code is available on Github. Feel free to join us!
